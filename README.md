Client Lead Management System (Mini CRM)

A lightweight CRM to manage client leads from website contact forms. Ideal for startups and small businesses to track leads, update statuses, and manage follow-ups efficiently.

🚀 Features

Dashboard Overview – See total leads, new leads, conversion rate, and last update.

Lead Management – View, update, and track leads with Name, Phone, Email, Source, and Status.

Lead Status Updates – Change status to New, Contacted, or Converted.

Notes & Follow-ups – Add notes and schedule follow-ups for each lead.

Secure Admin Access – Only authorized users can access and manage leads.

Responsive Frontend – Built with React.js, HTML, CSS, and JavaScript.

Database Support – Store leads and notes using MongoDB or MySQL.

🛠️ Technology Stack

Frontend: React.js, HTML, CSS, JavaScript

Backend: Node.js

Database: MongoDB
